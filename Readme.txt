1. Download the necessary packages to your machine:
	Hadoop, postgres, etc 
	Streamparse 
	Tweepy
	PsycoPG

2. Create Twitter develop profile and get access tokens

3. Download the file structure to your machine
 
4. Go into Twittercredentials.py and update Twitter credentials

5. Go into extweetwordcount > src > bolts > wordcount.py and update Twitter credentials

6. To run main program enter:
	sparse run (while in extweetwordcount folder)
	Hit Control C to stop program at any time 

7. To run finalresults.py enter:
	python finalresults.py <optional word you are interested in>

8. To run historgram.py enter:
	python histogram.py <number1>, <number2>
